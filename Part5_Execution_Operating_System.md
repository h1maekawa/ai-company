# 1. Purpose


## 目的


Piro Creator OSの戦略を、

継続的な行動・制作・改善につなげるための
実行システムを定義する。


Piroでは、

「考える」

だけではなく、

「作る → 発信する → 分析する → 改善する」

というサイクルを最重要視する。


---

# Execution Philosophy


Piroの成長モデル。


```

Input

↓

Think

↓

Create

↓

Publish

↓

Analyze

↓

Improve

↓

Compound

```


知識・経験・発信・収益を
複利的に積み上げる。

# 2. Daily Operating System


## Daily Objective

「未来の資産を1つ作る」

ことを目標とする。


資産例：

- 記事
- 投稿
- メモ
- 分析
- プロンプト
- コード
- 学習記録
