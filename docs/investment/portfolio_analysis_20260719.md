基準日: 2026-07-19  
対象: 資産残高、米国株式取引履歴、投資信託取引履歴

## 結論

現在の投資スタイルは、**インデックス投信を土台に、米国の成長株を集中保有する「コア・サテライト型」**である。

- コア: 投資信託 68.50%
- サテライト: 米国個別株 31.39%
- 個別株部分ではNVDAとMUが87.69%を占め、半導体への確信が強い。
- 過去の売却は短期売買よりも中期・長期の利益確定が中心。定義上の短期（数日〜2か月以内）で完結した売買は確認できなかった。
- NVDAの半分売却は会社の資金需要、MUの半分売却は700ドルでの利確目的だった。同じ「半分売却」でも理由が異なり、**資金確保のための売却と、投資判断による利益確定を分けて評価する必要がある**。
- 証券口座内の現金は2,085円、総資産の0.10%。楽天銀行残高は未取得のため全体の生活防衛資金は判断できないが、証券口座だけを見ると急な資金需要は資産売却に依存する。

### 行動パターン

1. **短期トレーダーではない**  
   履歴上、2か月以内に完結した取引はない。本人の実績に合う初期設計は、中長期を主軸にし、短期機能は少額の検証枠として分ける形。

2. **同じ部分売却でも、NVDAとMUでは意味が違う**  
   NVDAは会社に資金が必要となり、本来はもう少し保有したかったが13株を売却した。これは投資仮説に基づく利確ではなく、外部の資金事情による現金化。MUは700ドルを利確水準として売却を試み、CSV上では2026-05-11に1株が約定し、109,742円を受け取っている。

3. **今まで確認できる米国株売却はすべて取得価格を上回る**  
   勝ち銘柄を売る判断はできている。一方、損失銘柄を保有し続ける傾向があるか、損切り規則が機能するかは今回の履歴だけでは評価できない。

4. **投信積立は継続力があるが、入金額は変化している**  
   2024年春〜2025年7月は概ね月5万円、2025年8月以降は月2万円、2026年5月以降は月1.5万円。意図的な配分変更か、収入・支出の影響かを確認する必要がある。

5. **投信の15万円解約にも二つの理由がある**  
   2026年1月の先進国株式15万円解約は、必要資金の確保と、投資信託から個別株へ比重を移したいという方針変更の両方が理由。したがって、単なる生活費の取り崩しとして扱わず、資金需要と資産配分変更を別項目で記録する。

### 修正後の投資家像

前川さんは「利益が出たら機械的に売るタイプ」ではない。長期で持ちたい確信銘柄を保有しつつ、次の二つの理由で売却するタイプである。

1. 会社または本人に現金が必要になったときの資金確保
2. MUの700ドルのように、十分に上昇したと判断した価格での利益確定

特にNVDAでは売却後に「もう少し持っておきたかった」という感覚が残っている。このため今後は、売却の正しさを利益額だけで判定せず、資金需要がなければ保有を続けたか、売却後に後悔が残ったかも振り返り対象にする。

## 急な資金需要への対応力

### 現状

- 証券口座内の即時現金: 2,085円
- 楽天銀行残高: CSVでは未取得
- 株式・投信は売却可能でも、価格変動と受渡日があるため「現金と同じ」ではない。
- したがって、**売れる資産は多いが、売らずに使える待機資金はCSV上ほぼない**。
- 実際に会社の資金需要によってNVDAを希望より早く売却しているため、これは仮想的なリスクではなく、すでに一度発生した事象である。

現在値を単純に1株あたりへ換算すると、売却単位のイメージは次の通り。

| 資産 | 最小単位の評価額目安 | 特徴 |
|---|---:|---|
| NVDA 1株 | 約32,936円 | 必要額に合わせて段階売却しやすい |
| KO 1株 | 約13,245円 | より細かく金額調整しやすい |
| MU 1株 | 約137,869円 | 残り1株のため、売却するとMU保有がゼロになる |
| 投資信託 | 金額指定解約が可能 | 受渡までの時間、基準価額の変動を考慮 |

これは売却推奨順位ではない。実際の判断では、必要金額、期限、投資仮説、売却後の集中度、再購入方針を同時に比較する。

### 更新後の資金・配分方針

- 今後は会社経費を個人で実費負担する可能性が低いため、会社用の待機資金10万〜30万円は常設しない。
- 過去に会社資金のためNVDAを売却した事実は、売却履歴として残す。
- 投資資産の目標配分は、投資信託50%・個別株50%。
- 毎月投資できる金額は固定せず、自作家計簿アプリ「Flow+」の収支・予定支出を基に判断する。

現在の保有商品2,054,100円だけを50:50にすると、目標は投資信託1,027,050円、個別株1,027,050円。現在の個別株645,514円との差は381,536円である。ただし、今すぐ投信を売却するのではなく、新たに投資できる月は個別株側へ寄せ、段階的に50:50へ近づける。

### 家計簿アプリ「Flow+」との連携

ダッシュボードURL: https://household-finance-smoky.vercel.app/dashboard

公開画面で確認できた機能:

- ホーム
- 収支
- キャッシュフロー
- 投資
- 設定
- Googleログイン
- Gmail等の読み取り権限を使ったカード利用通知メールからの支出自動記録

実際の家計データはログイン後に保護されているため、公開画面からは閲覧できない。投資AIはGmailを重複して読むのではなく、Flow+が整理した結果を読み取る。

システム配置:

- Flow+は家計データの収集・集計元として維持する。
- 投資分析・購入候補・売却候補・判断ログの本体は、個人用AI会社の投資部門へ実装する。
- Flow+と個人用AI会社は別プロジェクトのまま、認証付き読取専用APIで連携する。
- Flow+にある既存の「投資」画面は、必要に応じて概要表示または個人用AI会社への入口として利用し、分析本体は置かない。

```text
当月の投資可能額
= 現在使える個人資金
+ 次回判定日までの入金予定
- 支払確定済みのカード請求
- 固定費・予定支出
- 個人生活用として残す金額
```

計算結果が0円以下の場合は、その月は新規投資を行わない案を表示する。CSV上は直近の投信積立が月15,000円だが、これを毎月必ず出せる余剰金とはみなさない。

Flow+から投資AIへ渡す項目:

- 判定対象月
- 現在の利用可能残高
- 当月の確定収入・入金予定
- 当月の確定支出
- 未引落しのカード利用額
- 固定費と一時的な予定支出
- 投資済み金額
- 投資可能額
- データ最終更新日時
- データ完全性と判定信頼度

## 月20万円という目標の整理

月20万円は年間240万円。現在の総資産2,056,185円に対して年間116.72%の取り崩し・利益が必要になる。現在資産から毎月安定して生む目標としては扱わず、**将来の必要元本を作る長期目標**と、**臨時の利益確定**を分けるべきである。

単純な必要元本の逆算例:

| 年間キャッシュ化率の仮定 | 月20万円に必要な元本 |
|---:|---:|
| 3% | 8,000万円 |
| 4% | 6,000万円 |
| 6% | 4,000万円 |
| 10% | 2,400万円 |

これは利回り保証ではなく、目標設定用の算数。10%を毎年安定して現金化する前提は変動・下落を無視できない。現在の保有は高配当収入型より値上がり益重視であり、月次収入より資産成長に向いた構成である。

## 前川さん専用AI会社へ追加する要件

### 1. CSV自動診断

- 楽天証券CSVのCP932文字コードを自動判定
- 残高・米国株履歴・投信履歴の突合
- 株式分割、部分売却、外貨決済、ポイント利用を正しく処理
- 会話上の概算とCSV確定値の差を表示

### 2. 投資行動プロファイル

- 売買ごとの保有日数を計算し、短期・中期・長期へ分類
- 全売却と部分売却を区別
- 利益確定、損切り、買い増し、積立、資金化の傾向を記録
- 「本人が考えるスタイル」と「実際の行動」の差を表示

### 3. 投資理由台帳

銘柄・仮想ロットごとに次を保存する。

- 購入理由
- 想定期間
- 成長カタリスト
- 買い増し条件
- 仮説が崩れる条件
- 利益確定条件
- 急な資金需要時に売却してよい株数
- 売却後に再購入する条件
- 売却理由（会社資金・生活資金・利確・仮説崩壊・配分変更）
- 売却後の納得度・後悔度

### 4. Flow+連携・投資可能額エンジン

入力: Flow+が集計した残高、収入、確定支出、未引落しカード利用、予定支出、データ更新日時。  
出力: 今月の投資可能額と、その計算根拠。

確定ルール:

- 会社用待機資金10万〜30万円の確保は判定条件から削除する
- 毎月の投資額は固定せず、Flow+の当月データから自動計算する
- 投資可能額が0円以下の場合は新規投資を見送る案を表示する
- データが古い、カード請求が未取得等の場合は信頼度を下げ、具体的な購入金額を確定しない
- 投資可能額がある場合、現在不足している個別株側へ優先配分し、投信50%・個別株50%へ近づける
- 個別株へ配分可能でも、銘柄別上限・半導体集中度・購入条件を通過しなければ購入候補にしない

### 5. 中身ベースの集中度

- 個別株だけでなく、投信の最新組入銘柄まで展開する。
- NVDA、MU、半導体、米国、成長株等の実質比率を計算する。
- 同じ企業を個別株と複数ファンドで重複保有している場合に警告する。

### 6. 期間別スコア

- 短期: RVOL、売買代金、スプレッド、値動き、決算・ニュース
- 中期: 2〜3四半期の業績、業績予想、業界サイクル、バリュエーション
- 長期: 市場成長、競争優位、財務、キャッシュフロー、仮説崩壊条件
- 同じティッカーでも3つのスコアと資金枠を別管理する。

### 7. 目標ギャップ管理

- 月20万円を「配当」「売却益」「元本取り崩し」に分ける。
- 12か月移動平均で実現損益を表示し、単月の好成績を給与のように扱わない。
- 目標達成に必要な元本、積立額、想定期間を複数シナリオで表示する。
- 最大下落率を含む悲観シナリオを必ず併記する。

## 投資判断エンジン初期ルール（2026-07-19確定）

詳細な購入・保有・売却・集中管理・検証ルールは、`12_FUND_POLICY_ENGINE.md`へ分離した。

- 個人生活用バッファ: 直近90日間の変動生活費を30日分に換算し、1.2倍する。
- 1銘柄: 個別株内30%で注意、40%で追加購入を原則停止する。
- 1テーマ: 個別株内60%で注意、70%で同テーマの追加購入を原則停止する。
- 現在のNVDA・MU合計87.69%は半導体テーマ上限を超えるため、初期状態では半導体の新規購入を原則停止する。ただし集中だけを理由に自動売却しない。
- 短期提案は、30日または20件の決着までpaper modeで検証する。
- 信用、レバレッジ、空売り、オプション、自動注文は初期対象外とする。
- AIは期間別スコア、購入上限額、株数、反対材料、撤退条件、データ時点を表示する。
