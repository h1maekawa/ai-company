# Company Monetization Map

## 1. 収益化システム全体図

```
[集客チャネル]                     [教育・誘導チャネル]                 [マネタイズポイント]
  - note (毎日投稿, SEO, バズ)  ───>  - 公式LINE (ステップ配信)  ───>  - 自社教材 (投資・副業・採用)
  - SNS (X / インスタ)          │     - 個別相談・面談         │      (価格帯: 5,000円〜30,000円)
  - SEOブログ                   └───>  - 記事内アフィリンク    └───>  - 高単価キャリア相談アフィリエイト
                                                                      (成果報酬: 10,000円〜50,000円/登録)
                                                                    - 投資口座開設アフィリエイト
                                                                      (成果報酬: 5,000円〜15,000円/開設)
```

## 2. マネタイズ・ファネルと収益シミュレーション

### パイプラインA: キャリア相談アフィリエイト (最優先)
- **フロー**: note記事（「未経験から優良IT企業へ転職する方法」等） -> エージェント紹介記事 -> 公式LINEでの推薦 -> 無料キャリアカウンセリング登録。
- **目標想定単価**: 平均 20,000円 / CV
- **目標CV数**: 15件/月
- **月売上想定**: 300,000円

### パイプラインB: 投資初心者向け口座開設
- **フロー**: note記事（「新NISAつみたて設定の落とし穴」等） -> 証券会社比較記事 -> 口座開設。
- **目標想定単価**: 平均 8,000円 / CV
- **目標CV数**: 10件/月
- **月売上想定**: 80,000円

### パイプラインC: 自社デジタルコンテンツ販売 (中期実装)
- **フロー**: noteでの無料記事量産 -> LINEステップでの「超実践的マネーリテラシー講座」 -> 有料教材e-bookまたはオンラインサロンへの勧誘。
- **目標想定単価**: 9,800円 / 個
- **目標販売数**: 20個/月
- **月売上想定**: 196,000円

## 3. アフィリエイト案件の優先順位

1. **キャリア/転職/就活支援エージェント案件**: 成果発生地点が「無料登録」または「面談完了」であり、ユーザー負担がなくCVRが非常に高いため。
2. **証券会社/積立投資信託口座開設**: 金融系は単価が高く、新NISA人気など社会的需要が大きいため。
3. **副業支援ツール/スクール紹介**: 「自分で稼ぐ力を身につけたい」というアクティブ層へ訴求するため。

## 4. 収益安定化（LTV向上）の施策

- **クロスセル**: キャリア転職に成功したユーザーに対し、次は「投資（資産形成）」コンテンツをLINE経由でレコメンド。
- **リピート商材**: 就活・転職時のサポートを終えたユーザーへ、入社後の「生産性向上（AI活用術）」などのビジネスハック系教材を提案。
