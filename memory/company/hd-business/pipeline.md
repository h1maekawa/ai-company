## 📊 確度定義

| 確度 | 意味 |
|------|------|
| S（90%以上） | 契約書サイン済み・入金確認待ち |
| A（70〜89%） | 口頭合意済み・契約手続き中 |
| B（50〜69%） | 商談中・前向きな反応あり |
| C（30〜49%） | 接触済み・検討中 |
| D（30%未満） | 初回接触のみ・要フォロー |

## 📌 記入ルール

- 初回接触した案件はすべてここに記入する。

- 滞留（最終接触から14日以上）案件は翌朝の `bottlenecks.md` に転記する。
