## 📞 架電スクリプト（冒頭3秒）

```
「お世話になっております。○○の件でご連絡させていただきました。
今○分ほどよろしいでしょうか？」
```

## 🔄 フォローアップフロー

| タイミング | アクション | チャネル |
|----------|---------|--------|
| 初回架電翌日 | リマインドSMS/メール | SMS・メール |
| 3日後 | 再架電 | 電話 |
| 1週間後 | 最終フォロー | 電話・メール |
| 2週間以上 | 滞留案件としてbottlenecks.md転記 | — |

## 💬 商談トークスクリプト（骨格）

1. **現状確認**: 「今、○○についてはどのようにされていますか？」
2. **課題深掘り**: 「その中で一番困っていることは何でしょうか？」
3. **理想確認**: 「理想的にはどんな状態になりたいですか？」
4. **提案**: 「実は○○でその課題を解決されているお客様が多くて…」
5. **クロージング**: 「まず○○から始めてみませんか？」

## ✅ 受注チェックリスト

- [ ] 顧客の課題・理想・予算が明確になっているか
- [ ] 決裁権者と話しているか
- [ ] 競合比較の懸念を解消したか
- [ ] 締め切り・開始時期を確認したか
- [ ] 次のアクションを顧客と合意しているか
