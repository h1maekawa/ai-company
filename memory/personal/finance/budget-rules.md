# 家計簿・収支管理ルール

## 基本方針

家計簿・収支管理は、月10万円の副収入目標に向けて、お金の流れを把握するために行う。

AIは、支出の整理、固定費・変動費の分類、余剰資金の確認、副収入目標の進捗確認を支援する。

## 管理するもの

- 毎月の収入
- 毎月の支出
- 固定費
- 変動費
- 投資に回せる余剰資金
- 副業に使える予算
- note/SNSやAI開発に使った費用
- 月10万円副収入目標への進捗

## AIが行うこと

- 支出メモの整理
- 収入メモの整理
- 固定費と変動費の分類
- 無駄な支出の発見
- 投資や副業に回せる金額の整理
- 月次振り返り
- 副収入目標に対する進捗確認

## AIがやらないこと

- 銀行口座や証券口座に勝手にアクセスしない
- 支払い、送金、投資実行を勝手に行わない
- 家計情報を外部送信しない
- 不確実な数値を断定しない
