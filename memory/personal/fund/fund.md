# Fund OS — 投資思想コア

## 投資哲学

前川弘行の投資は「資産形成の自動化」と「サテライトによる積極的リターン」の二層構造で成立する。
感情による判断を排除し、ルールベースで再現性ある意思決定を行うことを最優先とする。

---

## 投資対象条件

### コア資産（長期・積立・崩さない）
- 全世界株式インデックス（楽天VT / eMAXIS Slim）
- 先進国株式インデックス
- 毎月定額積立。相場に関わらず継続。売却判断はしない。

### サテライト資産（個別株・機動的売買）
以下の条件をすべて満たす銘柄を対象とする：

**必須条件**
- 明確な成長テーマに乗っている（AI・半導体・ロボット・電力インフラ等）
- 時価総額 100億ドル以上（流動性確保）
- 決算で継続的な増収増益トレンド
- 機関投資家の保有増加傾向

**優先条件**
- 競合優位性（モート）が明確
- セクター内シェアトップ3以内
- アナリスト評価が「Buy」または「Strong Buy」多数

---

## NG条件（絶対に手を出さない）

- 赤字継続かつ黒字転換時期不明の銘柄
- 過度な負債比率（D/E > 3 かつキャッシュフロー悪化）
- 訴訟・規制リスクが顕在化している銘柄
- 出来高が著しく薄い（流動性リスク）
- SNSのみで話題になっているミーム株
- FOMOによる衝動買い（ルール外エントリー）

## 売却ルール

### 利確ルール
| 保有期間 | 利確水準 |
|---------|---------|
| 短期（〜3ヶ月） | +15〜25% |
| 中期（3〜12ヶ月） | +40〜80% |
| 長期（1年〜） | ファンダメンタルズ崩壊まで保有継続 |

### 損切りルール
| 保有期間 | 損切り水準 |
|---------|----------|
| 短期 | -7% で即損切り |
| 中期 | -10〜12% でルール損切り |
| 長期 | ファンダメンタルズ崩壊のみ売却（一時的下落は保有） |

### ファンダメンタルズ崩壊の定義
- 2期連続で売上・利益が予想を大幅下回る（-15%以上ミス）
- 競合優位性の喪失（主力製品・技術のコモディティ化）
- 経営陣の大規模交代・不祥事
- 主力事業のビジネスモデル破綻

## AI判断ルール

Fund Manager AIは以下のルールで動く：

1. **感情コメント禁止**: 「上がりそう」「下がりそう」の感情的表現を使わない
2. **数値ベース**: すべての判断に具体的な価格・パーセント・日付を提示
3. **必須フォーマット遵守**: 回答は8項目フォーマットで返す
4. **ルール外エントリー阻止**: ルール条件を満たさない場合は「条件未達」と明示
5. **ログ推奨**: 重要な売買判断は必ずinvestment-logへの記録を促す
