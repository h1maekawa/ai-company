# Note Department — 発信テーマ・ペルソナ定義 (Themes)

最終更新：2026-06-18

---

## 🎯 展開する5大発信テーマ

### 1. AI会社構築 (AI Company OS)
- **ターゲット**: 個人事業主、フリーランス、効率化を目指すビジネスパーソン
- **コアバリュー**: 生成AIを活用して「一人経営の会社」をシステム化する極意
- **キーワード**: プロンプトエンジニアリング、Next.js OS、業務自動化、AI秘書

### 2. 投資 (Personal Finance / Fund)
- **ターゲット**: 個人投資家、資産形成層
- **コアバリュー**: 再現性ある投資判断OS（Fund Rules）の構築と実践
- **キーワード**: 資産形成、個別株分析、半導体、リスク管理、ポートフォリオ

### 3. 労働×偉人 (Work & Historical Leaders)
- **ターゲット**: キャリアに悩む若手ビジネスパーソン、リーダー層
- **コアバリュー**: 歴史的偉人の思考や行動特性から現代の労働・マネジメント術を学ぶ
- **キーワード**: 偉人伝、マインドセット、労働観、戦略思考

### 4. キャリア教育 (Career Education)
- **ターゲット**: 就活生、転職検討者、キャリアアップ志望者
- **コアバリュー**: 主体的なキャリア設計、評価される自己分析と選考対策テンプレート
- **キーワード**: 自己分析、就活対策、キャリアアップ、面接対策

### 5. 事業構築 (Business Strategy)
- **ターゲット**: 起業家、新規事業担当者、副業スタートアップ層
- **コアバリュー**: MVPから実ビジネスへ育てるためのフレームワークと実行ロジック
- **キーワード**: MVP、顧客開発、マネタイズ、事業戦略
