# note/SNS テーマ

## 発信方針

前川弘行が、非エンジニアの立場からAI会社を作っていく過程を発信する。

発信は実名寄りの半匿名で行う。  
SEO的にキャッチーで、検索されやすく、かつリアルな過程が伝わる内容にする。

## 主要テーマ

- 非エンジニアがAI会社を作る過程
- ObsidianをAI社員の記憶にする方法
- Claude Codeを使った個人開発
- AIで自分のタスクや思考を自動化する方法
- AIを使って副収入を作る過程
- note/SNSをAIで自動化する方法
- 個人用AI秘書の作り方
- AIで投資メモや家計簿を整理する方法

## 想定読者

- AIを使いたいが、何から始めればいいかわからない人
- 非エンジニアだが、AIで業務自動化したい人
- Claude CodeやObsidianに興味がある人
- 自分専用のAI秘書を作りたい人
- 個人収益化にAIを使いたい人
- 副業やnote収益化に興味がある人

## 記事の方向性

- きれいな成功談ではなく、作っている過程をリアルに見せる
- 非エンジニア目線で、つまずいた点も書く
- 実際のMDテンプレート、要件文、運用ルールをコンテンツ化する
- 有料部分では、実際のテンプレートや手順を提供する
- 最終的にはAI会社構築テンプレートや相談導線につなげる
