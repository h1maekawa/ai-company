# 前川弘行 プロフィール

## 基本情報

- 名前：前川弘行
- 生年月日：2000年4月20日
- 年齢：26歳
- 現在は株式会社Crestixの役員として従事している
- 非エンジニアだが、AIやClaude Codeを活用して、自分専用のAI会社を構築したいと考えている

## 現在の状況

現在、Crestixで役員として働いている。  
一方で、個人としての収入源も作っていきたいと考えている。

そのため、会社の業務とは別に、個人で副収入を作ることを重要なテーマとしている。

## 個人用AI会社を作る目的

個人用AI会社の目的は、前川弘行個人の収益・時間・学習・資産形成を支援すること。

特に、以下を実現したい。

- note/SNS発信を通じて個人収益を作る
- AIを使ってnote作成や発信作業を自動化する
- 投資メモや投資判断の振り返りをAIに支援してもらう
- 家計簿や収支管理をAIに整理してもらう
- 将来的には、自分が常に働かなくても、AIや資産が収益を生む状態を作る

## 第一目標

まずは、3ヶ月以内を目安に、個人として月10万円の副収入を作ることを第一目標にする。

収益源の候補は以下。

- note
- SNS発信
- 有料テンプレート
- 個人アプリ
- AIサービス
- 株式投資

ただし、投資については利益保証ではなく、AIには判断材料整理、リスク確認、振り返り、記録管理を支援してもらう。

## 性格・行動特性

- 面倒くさがりなところがある
- 飽き性なところがある
- ただし、仕組み化されていて、継続する意味が見えるものは続けやすい
- AI学習や新しい知識を得ることは好き
- 自分で全部手作業するより、AIに任せる仕組みを作ることに関心がある
- 難しい技術を細かく覚えるより、AIに正しく指示して成果物を作ることを重視する

## AI会社の使い方

個人用AI会社は、毎日15分程度、日記のように入力して運用する。

毎日の入力内容は、以下を想定する。

- 今日やったこと
- 気になったこと
- note/SNSのネタ
- 投資で気になった銘柄やニュース
- 支出や収入に関するメモ
- AI会社やClaude Codeで進めたいこと

AIは、それらを整理して、次の行動に変換する。

## AIに求める支援

- 雑なメモを整理してほしい
- 今日やることを優先順位付きで出してほしい
- note/SNSのネタ出し、構成、下書きを支援してほしい
- SEO的にキャッチーなタイトルやテーマを提案してほしい
- 投資メモを整理し、判断材料とリスクを分けてほしい
- 家計簿や収支のメモを整理してほしい
- Claude Codeに渡せる要件文を作ってほしい
- 技術的な内容は、非エンジニアでも理解できるように説明してほしい
- 面倒くさがりでも続けられるように、作業を小さく分解してほしい
